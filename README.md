Event Manager (Database Project)
Overview
Event Manager is a simple database-driven web application that allows  to add and view ,updateand delete (CRUD) events. Events entered through an HTML form are stored in a MySQL database (XAMPP) and displayed in a structured “View Events” section.

Features
Add new events via an HTML form (title, date, venue, description).

Store event data securely in MySQL.

Display all events in a clean, organized list.

CRUD operations add ,delete, update 

Demonstrates integration of front-end forms with backend database.

Tech Stack
Frontend: HTML, CSS,php

Database: MySQL (XAMPP)

Backend Integration: Guided PHP scripts for database connectivity

How to Run
Import the provided SQL file into phpMyAdmin to create the eventdb database and events table.

Place project files in the htdocs folder of XAMPP.

Start Apache and MySQL from the XAMPP control panel.

Access add_event.html to create events.

Navigate to view_events.php to see the list of events.
